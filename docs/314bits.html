314 bits

I made this the first drawings of this typeface back in 2009. Back then I just drew a couple of letters for a personal project. Some time passed, the year was 2014. I wanted to revisit this project and turn it into a complete font so I can use it on my website. I drew some more letters, but never finished the job.

Now it is 2019. It has been 10 years in the making, but I am very happy with the result. 314 bits is complete.

JWAX

JWAX is handcrafted wax for skate, snow, and surf. Manufactured by four friends in a basement in downtown Sofia, Bulgaria. This project is really special to me. Not only that I'm one of the owners, but I've unlimited creative freedom. It's been and still is an awesome journey, designing a product from the ground up.

Special thanks to everybody involved.
